# Creation of a CMake package.

This will create the following structure for a package called `libexample`
```
    INSTALLATION_DIR
    |
    |- libexample\
        |- lib\
           - libexample.lib (windows)
        |- include\
           - libexample\
              - libexample.h
        |- share\
           - libexampleConfig.cmake
           - libexampleConfigVersion.cmake
```
